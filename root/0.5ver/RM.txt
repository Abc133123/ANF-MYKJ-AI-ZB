# ANF-MYKJ-AI-VUP
1. 项目概述
项目名称：ANF-MYKJ-AI-VUP
项目类型：AI虚拟主播 (AI VUP)
开发者状态：新手开发者，个人项目
项目状态：开发停滞，兴趣转移
最新版本：ver0.5 (位于root目录下)
哔哩哔哩视频:https://www.bilibili.com/video/BV1GUQ6BYEHK/

2. 技术栈与架构
这是一个集成了多个技术的本地化AI主播系统：
大语言模型：调用 Deepseek API 进行对话生成。
语音合成：使用 VITS-GPT (即 VITS + GPT 相关的TTS模型) 生成语音。
2D模型驱动：通过 VTS (VTube Studio) 包来控制 Steam 上的 VTube Studio 软件，驱动 Live2D 模型。
直播推流：计划接入哔哩哔哩直播，目前仅为本地+API调用模式。

3. 当前实现功能 (ver0.5)
成功实现了 Live2D 模型的基础控制（被描述为“live2d瞎看”，即能做出一些随机或基础的响应动作）。
搭建了本地与 Deepseek API 交互的基础链路。

4. 主要问题与痛点
直播集成困难：开发者认为哔哩哔哩直播的“饼干”（可能指弹幕监听、登录验证或相关SDK）研究起来很麻烦，且每次登录都需要刷脸验证，体验不佳。
开发动力不足：觉得项目“不好玩”，兴趣点已转移到其他方向。
项目形态原始：目前只是一个本地应用+API调用，距离完整的AI主播还有较大差距。

5. 未来方向 / 开发者新兴趣
开发者已暂停此项目，计划探索：
Unity 手机端接口：可能想将AI能力移植到移动端或Unity引擎中。(毕竟谁家用电脑本地 不是服务器网页就是手机app要么硬件瞎玩)
其他有趣的应用：寻找更吸引人的技术方向。
